Web Dizajn

Pitanja i odgovori

1.Šta je internet?

Internet je svetski sistem umreženih racunarski mreža.

2.Kad i kako je nastao internet?

Nastao je 1969,stvaranjem mreže racunara pod kontrolom USA.

3.Šta je veb?

Samo jedna od sredstava za razmenu informacija preko interneta.

4.Kako je nastao veb?

Tim Berners-Li je osmislio sistem upravljanja informacijama sa konceptom hiperteksta.

5.Šta je veb sajt?

Skup veb stranica koje mogu da sadrže razne multimedijalne sadržaje.

6.Kako se pristupa veb stranicama?

Veb stranama se pristupa preko internet browser-a pomocu URL-a.

7.Šta je HTML?

Jezik za pravljenje stranica pomocu tagova.

8.Šta su internet servisi?

Predstavljaju razlicite vrste usluga interneta.Primeri:

  1. WWW - prikaz HTML stranica
  2. E-mail - slanje i primanje e-pošte
  3. FTP - prenos datoteka sa servera na server

9.Objasni klijent servera.Najznacajniji klijent-serveri?

Klijent traži usluge,server obezbedjuje resurse i usluge.www i e-mail

10.Objasni P2P.

Više kompjutera povezani preko interneta.

11.Navedi korake kreiranja staticke veb stranice.

  1. Kreiranje strane,hostovanje
  2. Zahtevanje strane
  3. Server nalazi stranu
  4. Server vraca stranu
  5. Prikazivanje strane

12.Navedi korake kreiranja dinamicke veb stranice.

U kucistu se nalazi:

  1. Kreiranje strane,hostovanje
  2. Zahtevanje strane
  3. Server nalazi stranu
  4. Server izvršava kod
  5. Server vraca stranu
  6. Prikazivanje strane

13.Cemu služi DNS?

Sistem koji pretvara imena racunara u IP-adrese.

14.Objasni domene u mat.bg.ac.rs

  1. mat - ???
  2. bg - Bugarska
  3. ac - Ostrvo u Velikoj Britaniji
  4. rs - Srbija

15.Šta je URL?

Jedinstvena adresa na internetu.

  1. http:// - protokol
  2. www.mojdizajn.com - sajt
  3. /2008/primeri/prvi.html - putanja

16.Šta radi server kada dobije zahtev za ime direktorijuma

Prikaže default.html

17.Šta je registracija domena.Navesti nacionalne domene

Dobijanje veb adrese bez prostora za postavljanje sajta.

  1. rs
  2. ac
  3. bg

18.Šta su registri domena i ko ih ovlašcuje?

To su kompanije koje se brinu o registraciji domena.Njih ovlašcuje ICANN– Internet.

19.Šta je RNIDS?

(Registar nacionalnog internet domena Srbije).

20.Navedi sve potrebne korake za registraciju domena.

  1. Osmisliti dobar i dozvoljen naziv domena
  2. Proveriti da li je željan naziv domena slobodan
  3. Izabrati pravi domenski prostor
  4. Registracija domena preko nekog ovlašcenog registara

21.Šta je veb hosting?

Smeštanje stranice na veb server.
